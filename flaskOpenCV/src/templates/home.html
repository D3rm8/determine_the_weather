Face Detection

Project Description

This is a basic face detection project which uses OpenCV and the Python programming language to detect faces in images

Project Details

  • User uploads image from file
  • The image is then processed using a HAAR Cascades files
  • Faces are detected using a HAAR Cascades file
  • An image is displayed with a square drawn around detected faces
  • The number of faces detected is also displayed
